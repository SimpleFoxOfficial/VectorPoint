## How To Install
1. Download Modrinth Launcher / Prism Launcher - Whatever takes .mrpack (modrinth modpack file)
2. Set your launcher up
3. If you use Prism:
   Drag&Drop .mrpack in the launcher and just click OK.
   If you use Modrinth:
   Click "Add Instance", then choose "Import" and select .mrpack
4. Wait until it downloads and make sure you installed the correct modpack
5. Launch it and enjoy the gameplay!
